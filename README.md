# HR Attrition Machine Learning Solution

![enter image description here](https://github.com/RNagendra12/hr-employee-attrition/blob/main/Attrtion.png?raw=true)

Details about the project - This project is based on HR data which is from a random organization. It helps by analyze the employee data and detect employees that are about to leave the organization.

The project was created as a part of Keggle Competition. This data contains past employee data of 15 years from HRMS.

It is a classification based project for which the output is either employee will leave or not.
